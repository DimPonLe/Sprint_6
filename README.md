# Sprint_6
# Эта практическая работа в рамках обучения в "Яндекс Практикум"

# Вся работа пока ведется в ветке develop на период проверки

# Этот учебный проект посвящён автоматизации тестирования сервиса по заказу самокатов с помощью паттерна Page Object Model

## Архитектура проекта
- `tests/` — папка с тестами.
- `pages/` — папка с классами страниц.
- `locators/` — папка с локаторами элементов.
- `conftest.py` — фикстуры и настройки PyTest.
- `requirements.txt` — библиотеки и модули

## Тестирование
- Для запуска тестов используйте PyTest.
- Отчет ведется с помощью Allure.
